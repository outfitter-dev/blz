BLZ Agent Instructions

Purpose
- Local, fast search over llms.txt/llms-full.txt with exact line ranges.

Core Commands
- Add source (non-interactive): blz add <alias> <llms.txt-url> -y
- Update all sources:           blz update --all
- List sources (JSON):          blz list -o json

Search
- All sources:                  blz "<query>" -o json
- By source:                    blz "<query>" -s <alias> -o json
- Pagination:                   --limit <N> --page <P> | search --next | --prev

Search JSON fields (typical)
- alias, lines (e.g. "120-142"), snippet, score, headingPath, sourceUrl

Get Exact Lines
- Fetch text:                   blz get <alias> --lines <range> [--context N]
- Ranges:                       "A-B" | "A+N" | "A:B,C:D"

Agent Flow (recommended)
1) Ensure source exists (blz list -o json); if missing, blz add <alias> <url> -y
2) Search for topic (blz "query" -s <alias> -o json)
3) Pick top result; extract alias + lines
4) Fetch content (blz get <alias> --lines <lines> --context 3)

Notes
- Prefer llms-full.txt when available.
- Exit codes: 0=ok, 1=user error, 2=system error.
