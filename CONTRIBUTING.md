## Quick Start

```bash
# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Clone and build
git clone https://github.com/outfitter-dev/blz.git
cd blz
cargo build --release

# Run tests
cargo test --workspace
```
