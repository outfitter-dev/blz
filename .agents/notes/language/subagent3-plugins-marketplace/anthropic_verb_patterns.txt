MULTILINGUAL VERB PATTERNS - PLUGIN & MARKETPLACE TERMINOLOGY
==============================================================

1. INSTALL / ADD OPERATIONS
===========================

English:
  - Install plugins
  - Add marketplaces
  - Manage plugins

French:
  - Installez les plugins (imperative: install)
  - Ajoutez des marketplaces (imperative: add)
  - Gérez les plugins (imperative: manage)

German:
  - Plugins installieren (infinitive: to install)
  - Installation überprüfen (noun: installation check)
  - Marketplaces hinzufügen (infinitive: to add)

Portuguese:
  - Instale plugins (imperative: install)
  - Adicione marketplaces (imperative: add)
  - Gerencie plugins (imperative: manage)

Italian:
  - Installare plugin (infinitive: to install)
  - Aggiungere marketplace (infinitive: to add)
  - Gestire le operazioni (infinitive: to manage)

Indonesian:
  - Menginstal plugin (infinitive: to install)
  - Menambahkan marketplace (infinitive: to add)
  - Mengelola operasi (infinitive: to manage)

4. KEY LANGUAGE INDICATORS
===========================

LANGUAGE IDENTIFIERS BY MORPHOLOGY:

French:
  - -ez endings (vous form imperatives): Installez, Ajoutez, Gérez
  - le/la articles: les plugins, des marketplaces
  - Accented characters: é, è, ê, ç
  - Words: étendez, configurez

German:
  - -en endings (infinitives): installieren, entwickeln
  - Capital nouns: Plugins, Marketplaces
  - -ieren verbs: organisieren, konfigurieren
  - Compound words: Plugin-Marketplaces, Marketplace-Verwaltung

Portuguese:
  - -e endings (imperatives): Instale, Adicione, Gerencie
  - Nasal sounds (ã, õ): marketplaces
  - Words: gerencie, desenvolva, compartilhe
  - Tilde accents: ã, õ

Italian:
  - -are/-ere/-ire infinitives: installare, aggiungere, gestire
  - -i plurals: dei plugin, dei marketplace, i metadati
  - Dell'/dei articles: Marketplace dei plugin, dei creatori
  - Words: creare, verificare, rimuovere

Indonesian:
  - Meng- prefix (infinitive aspect marker): Menginstal, Menambahkan, Mengelola
  - -kan suffix: Memperbarui, Menghapus
  - No inflections (isolating language)
  - Words: buat, kelola, daftar

7. STRONGEST LANGUAGE INDICATORS
=================================

FRENCH:
  - -ez verb endings (Installez, Ajoutez, Gérez, Développez)
  - les/des articles
  - qu'ils/qu'elles pronouns
  - Accented vowels (é, è, ê, ç, à)

GERMAN:
  - Capital nouns (Plugins, Marketplaces, Verwaltung)
  - Infinitive -en endings (entwickeln, organisieren, teilen, testen)
  - Compound words with hyphens or concatenation
  - ü/ö/ä characters

PORTUGUESE:
  - Imperative -e endings (Instale, Adicione, Gerencie)
  - Nasal vowels (ã, õ)
  - Words like "desenvolva", "compartilhe", "próximos"
  - Tilde accents

ITALIAN:
  - Infinitive -are/-ere/-ire endings
  - -i/-e plurals (dei plugin, dei marketplace)
  - Articles: il/la/dello/della
  - Words like "creare", "verificare", "rimuovere", "ospitare"

INDONESIAN:
  - Meng- prefix (Menginstal, Menambahkan, Mengelola)
  - No inflections or articles
  - Words like "buat", "daftar", "kelola"
  - Ang-ism patterns (Ang... = to... structure)
